# Smart-Resource-Manager
Smart Resource Manager: Allocation &amp; Utilization Suite
***Overview***
Smart Resource Manager – Allocation & Utilisation Suite is a responsive, front-end dashboard application designed to monitor, allocate, and analyze organizational resources across teams and projects. It provides real-time visibility into resource utilization, project progress, and performance metrics through an intuitive UI and interactive visualizations.
The suite is ideal for engineering managers, project leads, and resource managers who need a clear, consolidated view of capacity planning and workload distribution.

***Key Features***
1. Dashboard & KPIs
- Total Resources
- Active Projects
- Average Utilization
- Available / Unallocated Resources

2. Data Visualization
- Line Chart: Overall resource utilization over recent months
- Bar Chart: Project efficiency comparison
- Doughnut Chart: Resource distribution across teams
 ***(Implemented using Chart.js)**

3.Project Management
- View ongoing projects with:
- Team assignment
- Progress percentage
- Due dates
- Add new projects dynamically via modal form
- Automatic progress bar rendering and card generation (client-side)

4. Team & Allocation Insights
- Top team allocations with utilization percentages
- Visual progress indicators for quick assessment

5. User Profile Page
- User details (name, role, department, location)
- Profile avatar
- Actions for edit profile and logout
- Seamless navigation back to the dashboard

***Tech Stack***
HTML5 – Page structure and layout
CSS3 / Bootstrap 5.3 – Responsive styling and UI components
Font Awesome 6 – Icons and visual cues
JavaScript (Vanilla) – Interactivity and DOM manipulation
Chart.js 4 – Data visualization and analytics
